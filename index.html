Week Six - 05/11/2023

This week, I had to migrate my static assets away from Google cloud platform.

This involved the following actions:

  1. I disabled my Google Cloud Platform billing account
  2. I migrated the static assets to Cloudflare R2
  3. I set up a CORS policy on the bucket to allow cross-origin requests to it
  4. I rewrote all the old url paths to the new R2 public URLs.

I also wrote CSS to make the site mobile first as so:

  1. I made the heading collapse into flex-display column below a mobile breakpoint
  2. The main content is mobile safe using a fixed width as to keep consistency
  3. I hid the table on mobile viewports due to its inflexibility
  4. All relevant meta tags are included in the head

On top of this, I included these changes:

  1. I organized all CSS files into independent folders
  2. All images are flexible and will not overflow their containers (max-width)

Week Four - 22/10/2023:

This week, I created custom CSS for the webpage in my favourite style reminiscent of the 2000s web (or 'webrevival' affectionately)

  1. I added a background image
  2. I used flexbox to create a responsive navbar
  3. I removed bullet points and the margin they add
  4. I implemented a custom font using the optimised for web format WOFF with @font-face
  5. I used colour to highlight certain elements like spans and anchors
  6. I added media queries to fix the navbar and content window on mobile viewports
  7. I added the relevant meta tags to keep content working on mobile
  8. I moved the table inline style to CSS
  9. I minified normalize and included its source map to speed up load times
  10. I added suitable font fallbacks to ensure relevant fonts are always applied
  11. I specified font-display in @font-face to make sure that on downloading the font, the load will be non-blocking (using fallback)
